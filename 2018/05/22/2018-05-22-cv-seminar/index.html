[计算机视觉]CV讨论班

发表于 2018-05-22
研究生师兄弟间组织的、关于计算机视觉的讨论班的笔记整理，主要是读懂论文和跑通代码。

讨论班概况

主题

CV: paper + code
代码上尽可能围绕PyTorch展开。

Research方面以CV方向的Paper Reading为主。
Engineering方面以围绕Research做相应展开。
Topic包括但不限于Face，Detection，Segment，Re-ID，pose estimation等方面。

T字形战略

先橫后纵，在保证广度的前提下，发展深度。

CV四个典型任务

  • Classification（分类）
  • Detection（检测）
    • One-Stage：YOLO、SSD
    • Two-Stage：R-CNN系列（R-CNN、Fast R-CNN、Faster R-CNN）
  • Segmentation（分割）
    • Semantic：DeepLab
    • Instance：Mask R-CNN
  • Tracking（追踪）
  • 模型压缩与加速

学术界现状

  • Detection：大咖在刚
  • Segmentation+Tracking：可能是我们可以做的方向
  • Classification：已经基本没人做了
