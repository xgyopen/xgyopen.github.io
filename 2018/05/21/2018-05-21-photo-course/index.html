*[拍照]手机拍照及修图技巧

发表于 2018-05-21
一门手机拍照课程的笔记。包含后期修图。

§1 你好，手机摄影！手机摄影必备核心知识大搜罗（手机基础拍照知识）

对焦功能

  1. 什么是对焦
    对焦也叫对光、聚焦。通过相机内的对焦机构，变动物距和相距的位置，使被拍物成像清晰的过程就是对焦。

  2. 如何对焦
    手机对焦：点击屏幕
    日常拍照基本要求：把主体拍清晰（对焦准确）

应用：虚实对比照

应用：虚化光斑拍摄

故意制造虚焦效果：夜晚平常不起眼和杂乱的场景 → 截然不同的虚化感觉

手机虚焦方法一：专业模式虚焦法（适合有专业模式的手机，如OPPO、VIVO、华为、小米等）

  1. 打开手机相机，准备拍摄。
  2. 选择进入相机的专业模式。
  3. 找到焦距选项，横向拨动焦距滑杆，清晰或是模糊我们可以轻松改变。

手机虚焦方法二：全能虚焦法（适合没有专业模式的手机，如iPhone6s等）

  1. 打开相机，这时候画面中的灯都是实焦的。
  2. 让小伙伴把手放到手机镜头前很近的地方，我们点击屏幕对手部进行对焦，然后做一个关键操作，用手指长按屏幕1.5秒左右，会锁定对焦。
  3. 我们把手移开，放大画面就会看到，原来实焦的灯光变虚了。

应用：错位照拍摄

  1. 手机尽量保持在低角度（台阶：从下向上拍）
  2. 设置对焦点在后景人物身上
  3. 积累错位照创意

测光操作

  1. 亮度调节
    手机上下滑动，增减亮度

  2. 大光比环境
    画面亮部暗部对比非常强烈

  3. 应对大光比
    遮挡+道具

  4. 常见光线角度（顺光、逆光、测光、顶光、底光）

①顺光
顺光也叫做“正面光”，指光线的投射方向和拍摄方向相同的光线。顺光时，被摄体受到均匀照明，影调比较柔和。

②逆光
逆光是一种由于被摄主体恰好处于光源和照相机之间的状况，在一般情况下摄影者应尽量避免在逆光条件下拍摄物体，但是逆光情况有时也可用于某些艺术创作。

Tips漂亮的逆光发丝
打开手机闪光灯作为逆光光源，移动到人物的正后方，人物的头发会被打亮

Tips逆光一寸照
逆光+黑白滤镜

应用：逆光剪影照

室外：日落时，让被摄主体背对太阳 → 人物黑，天空亮 → 剪影风格图片

室内：朝阳的窗户，剪影着重强调被摄者的曲线或形态

逆光剪影Tips

  1. 逆光拍摄，让主体背光
  2. 日出日落时是理想的拍摄时间
  3. 被摄者后面的强烈光线，使前景与背景的亮度相差非常大，形成剪影
  4. 拍摄剪影，着重强调被摄者曲线或形态
  5. 窗边、山顶、海边、城市 街道
  6. 手机拍摄剪影只需要在适当环境下，对着背景测光，被摄者就会变黑

③侧光
侧光，即来自景物左侧或右侧的光线，同景物、照相机成90度左右的水平角度。这种光线能产生明显的强烈对比。

“质感照明”：把光源移动到人物测面，测光使被摄主体一测受光，产生强烈的明暗对比，上下滑动屏幕，调节亮度，找到最酷的效果

④顶光
顶光，即来自顶部的光线，与景物、照相机成90度左右的垂直角度。人物在这种光线下，其头顶、前额、鼻头很亮，下眼窝，两腮和鼻子下面完全处于阴影之中，造成一种反常奇特的形态。

因此，一般都避免使用这种光线拍摄人物。(骷髅光)

⑤底光
底光，即来自底部的光线，从下至上与景物、照相机成90度左右的垂直角度。一般底光会有阴森恐怖的感觉。

因为再高的颜值也架不住底光的摧残。

    自拍时：大拇指按下音量键快门

应用：高速慢速快门

视频录制

应用：短片拍摄剪辑

视频神器：VUE

VUE拍摄短片的过程：

  1. 同拍照一样，先点击人物对焦
  2. 向左滑动屏幕，挑选一款我们喜欢的滤镜
  3. 引导模特就位，点击开始视频记录。视频拍摄，重在引导和沟通。

Tips朦胧画面技巧

全景模式

Tips全景分身照

  1. 打开手机的全景模式
  2. 手机缓慢水平移动
  3. 移动到模特时，手机停止，这时模特从摄影师身后绕开
  4. 模特绕到相机位置之外，摆好姿势
  5. 重复以上步骤，继续移动手机，搞定

注意：

  1. 不能让模特直接横着跑过去。
  2. 不要在拍全景的时候跳起来，不然会让人“灵肉分离”。
  3. 全景模式拍大长腿，会把腿拍的更短……

日系小清新风格特点

  • 高曝光
  • 低饱和
  • 淡色彩

常见参数

  1. 画面明亮-曝光
    曝光，是指被摄影物体发出或反射的光线，通过照相机镜头投射到感光片上，产生显影的过程。曝光度越高，画面越亮；曝光度越底，画面越暗。
    调节整体亮/暗

  2. 画面明亮-阴影补偿
    对暗部有针对性地提亮

  3. 画面柔和-对比度
    对比度，指的是一幅图像中明暗区域最亮的白和最暗的黑之间不同亮度层级的测量，差异范围越大代表对比越大，差异范围越小代表对比越小。
    调节整体反差，亮部更亮，暗部更暗
    想要小清新风格，需要降低对比度

  4. 画面柔和-高光减淡
    压暗高光亮度，使亮部没有那么亮，图片整体的反差减弱

  5. 色彩清淡-饱和度
    饱和度，指色彩的纯洁性，也叫饱和度或彩度，是“色彩三属性”之一。饱和度取决于该色中含色成分和消色成分（灰色）的比例。含色成分越大，饱和度越大；消色成分越大，饱和度越小。
    饱和度不宜大量增加，会显得颜色艳丽，色彩失真

  6. 色彩清淡-褪色
    轻松让色彩不饱和：使画面看起来像蒙上一层灰色，降低颜色的鲜艳程度，画面的反差也随之降低

修图流程

传统的修图流程：“滤镜——调整”

推荐的修图流程：“原始调整——添加滤镜——最终调整”

  1. 原始调整
    曝光、阴影补偿、对比度、高光减淡、饱和度、褪色、清晰度（增加局部的对比度）、锐化（增加边缘的对比度）
    PS：处理人像时注意不要增加太多清晰度，会使人物脸部变得很脏

  2. 增加滤镜
    A2

  3. 最终调整
    色温调整、增加暗角、阴影色调
    PS：对画面整体性进行的修饰操作（如色温调整、增加暗角）尽量留在最终环节

VSCO APP暗调城市风格调色攻略

暗调城市风格特点：暗色调、反差较大、颜色不鲜艳、比较厚重

  1. 原始调整
    降低曝光、增加对比度、增加褪色、增加清晰度、增加锐化、降低饱和度

  2. 增加滤镜
    F2滤镜/HB1滤镜/HB2滤镜

  3. 最终调整
    降低曝光、调整对比度、增加褪色、增加暗角

§3 身边不起眼的场景，也能作为拍照利器

环境构图：平凡场景拍照技巧

1、借助环境延伸构图
借助栏杆的延伸性，从栏杆的一端，低角度斜向去拍，这时延伸出去的栏杆刚好分割画面，简单的变换角度，本来不起眼的栏杆就起到了辅助构图的作用

PS：胳膊不要实实在在的放在栏杆上，而是可以藏在后面，用手腕轻轻倚靠栏杆，这样手臂就不会那么抢戏啦。

用线条状的事物分割画面：如头顶上普通的电线、站在高处从上向下俯拍人行道

2、发现环境新视角（楼梯俯拍）
拍照时要时刻记得借助环境去辅助构图，比如利用栏杆切割画面，俯拍+对角线分割构图，马上跳出常规构图！

练习构图的方法：比如想练习对角线构图，那么什么都不要想，这周只看、只拍对角线构图的图片，一周下来，必定会有收获。

3、借助前景遮挡构图
前景可以辅助构图，同时也能使画面有前后层次。

环境互动：平凡场景拍照技巧

1、与玻璃的创意互动（玩出猪鼻子）
引导模特把鼻子靠在玻璃上，这样就得到了一个长着猪鼻子的小伙伴。拍摄时，我们最好从正面平拍，以便突出猪鼻

2、融入场景，拍出情绪
小北的口诀：一杯咖啡，一本书，一个慵懒的午后。
当然了，我们最好让模特坐在窗边，借助玻璃，照片会更加文艺。

另一个比较容易拍出慵懒文艺照片的是居家场景。早上起床，伸个懒腰，开始美好的一天。

窗帘营造的朦胧效果，加上居家场景中的慵懒情绪，我们可以轻松拍出带有情绪的照片。
注意：由于从室内向室外是逆光拍摄，所以要点击屏幕对焦，然后向上拖动滑杆提升画面亮度，保证拍出的照片不会太暗。

还有一个办法，把自己设置成一个旁观者角色，让模特在某个环境中自由发挥，去展现自己最真实的一面，如果早晨的树林很冷，那么我们需要的只是按下快门，静静的去记录她的每一个表情和情绪。

小北推荐的两个日本摄影师：滨田英明、森友治

3、与摄影师互动（消除紧张感）
有趣的小技巧：告诉模特“你试试往玻璃上哈气，然后画个心出”。当她在画心的过程中，可以进行抓拍，这样，就不用再担心拍出不自然或者比较拘束的照片了。

让模特模仿有趣的动作，比如，在抓娃娃时，好不容易抓出一个娃娃，这时候，让小伙伴模仿娃娃的大耳朵，她可能也会忘记紧张。

发现生活：平凡场景拍照技巧

每天下午日落时刻，有房檐的建筑会因为夕阳西下，而产生阴影。

借助湖面的倒影进行拍照

托起一张白纸，可以起到反射光的作用

借助一个普通的“近视镜”，也可以达到类似的大光圈虚化效果
首先设置虚化对焦，得到一个模糊的画面，然后眼镜就可以帮助我们的手机缓解近视了。

只靠手机闪光灯，也可以拍出酷酷的肖像照
首先选择在晚上，保证环境足够黑，然后从侧面使用手机闪光灯照射，找到合适的光源位置即可，然后可以将图片处理成黑白，这样就获得了一张酷酷的肖像照。

附件技能点

餐厅摆拍口诀

牙疼、脸疼、左脸疼、右脸疼、全脸疼

树林小技巧：旋转抓拍

  1. 找到一个视野开阔、阳关灿烂的地点
  2. 模特就位，打开相机
  3. 让模特旋转起来，速度不用很快，但动作要连续
  4. 长按拍照键，进行高速连拍

制造阴影小技巧

  1. 单手抬高，让阳光透过指缝
  2. 模特闭眼，找到合适阴影位置进行拍摄
  3. 也可以借助道具制造阴影

小技巧：挖个圈圈包围你

  1. 早餐克扣一片面包做道具
  2. 将扣好圈圈的面包置于镜头前
  3. 调整位置，使人物脸部被圈圈包围
  4. 和她互动起来吧~

小技巧：人肉遮光术

拍照时，如果一侧阳光过强，人脸就会过曝，使面部亮度很不协调。

  1. 让一个小伙伴……脱掉毛衣…
  2. 将毛衣举起，遮挡过强的太阳光
  3. 观察人脸光线变化，人肉遮光术get
